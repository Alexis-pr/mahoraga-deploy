Mahoraga — Technical Interview Evaluation Platform

Mahoraga is a web-based technical evaluation platform designed to help software development candidates practice technical interviews. The system allows users to answer interview questions by technology and difficulty level and receive structured feedback generated by artificial intelligence at the end of each session.

The name Mahoraga represents continuous adaptation: just like the fictional entity that adapts to any technique, the platform adapts to the candidate's level and evolves its evaluation with every interview.

AI-Based Evaluation

At the end of each interview session, the platform sends all responses in a single request to OpenAI's GPT-4o-mini model.

The AI analyzes the answers and returns:

Individual scores per question

Explanations for each score

Identified strengths

Identified weaknesses

Personalized recommendations

A global motivational feedback summary

Progress Roadmap

Mahoraga includes a visual roadmap that tracks progress for each technology.

Text Input

Voice Recording

This functionality uses the MediaRecorder API.

AI-generated improvement recommendations

System Architecture

Mahoraga follows a classic client-server architecture with a clear separation between a static frontend and a RESTful backend.

Interview Flow

The user selects a technology and difficulty level from the roadmap.

The frontend creates a session in the backend using:

POST /api/interview

Five questions are retrieved randomly from the question database.

The user answers each question.

During the interview, responses are scored locally to avoid delays.

When the interview ends, all responses are sent to the backend.

The backend sends the evaluation request to OpenAI.

GPT-4o-mini returns structured evaluation data.

The results panel displays detailed feedback.
